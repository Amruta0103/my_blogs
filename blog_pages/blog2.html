For this week, I started working on different CSS properties. The key points of my learning were as folows :-
  • background-image,
  • slideshow,
  • sidebar, &
  • pagination.

background-img:
This property helps us to set an image or to set up a multi-color gradient in the background for the specified element. The gradient could be linear or radial, meaning in a straight-line or starting from center to outside in circular fashion, respectively.


Slideshow:
In many of the web pages or web applications we see that many images are displayed. These keep changing automatically or manually using “forward” and “backward” buttons. It is very useful and good looking component.

Sidebar:
This is a very useful component when you have multiple options of links where the user could be redirected. You can pull out the sidebar or it could be fixed at the side permanently. It helps to separate out the links or menu options for the convenience of the user.

Pagination:
This is the component that tells us the page number of the document, which is set up at the end of the page. Just like, at the end of search engines pages we see, how many more pages we can surf through for the information we need. A simple example for this would be the number of “o’s” at the end of Google’s tab.
